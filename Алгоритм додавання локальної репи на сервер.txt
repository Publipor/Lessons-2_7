1. git init - ініціалізація локального репозиторію
2. git log  - історія комітів (виправлень)
3. git status - історія змін локальних файлів
4. git add . - Додавання всіх файлів до локального репозиторію
5. git commit -m "Назва коміту" - створює на сервері опис змін і їх версію
6. git remote add origin https://github.com/Publipor/Lessons-2_3.git  - додавання віддаленого репозиторію до  локального
7. git push -u origin master - завантажує локальний репозиторій на сайт
