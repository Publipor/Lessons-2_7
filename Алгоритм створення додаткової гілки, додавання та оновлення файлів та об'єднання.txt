1. git init - ініціалізація локального репозиторію
2. git status - історія змін локальних файлів
3. git checkout -b New - створення нової гілки
4. git add new_file1.txt new_file2.txt new_file3.txt new_file4.txt new_file5.txt "Алгоритм створення додаткової гілки, додавання та оновлення файлів та об'єднання.txt"
 - Додавання всіх файлів до локального репозиторію
5. git commit -m "Назва коміту нової гілки" - створює на сервері опис змін і їх версію
6. git remote add origin https://github.com/Publipor/Lessons-2_3.git  - додавання віддаленого репозиторію до  локального
7. git push -u origin New - завантажує локальний репозиторій на сайт 


------------------------------------------------------------------------------- Оновлення репозиторію з локальний в мережу

1.1 git status - історія змін локальних файлів
1.2 git add . - Додавання всіх файлів до локального репозиторію
1.3 git commit -m "Опис змін нової гілки" - Опиз змін
1.4 git push origin New - заливання змін в репу
